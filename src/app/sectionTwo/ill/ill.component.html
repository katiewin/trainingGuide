ILL

For efficient handling of interlibrary loan requests, familiarize yourself with the ILLiad procedures outlined below. Ensure a smooth process for checking out and checking in items with green slips using the ILLiad web circulation system. Follow the steps diligently to provide prompt and accurate service to patrons.

ILLiad Procedures

Items Eligible: Only those with green slips.

Access ILLiad Web Circulation: Open the ILLiad web circulation bookmarked on your browser.

Password Renewal: If the password has expired, check the mailroom for the new password.

Checking Out Books: Open ILLiad, click the empty box next to "Check Out," then scan and desensitize the item. Hand it to the patron.

Checking In Books: Open ILLiad, click the empty box next to "Check-In," then scan and sensitize the item. Place it in the courier bin.
